Run the Java Applet:

  • It is intended for students to practice identifying functions (graph --> equation)
  • If you want to graph equations (equation --> graph) then please download Graphmatica.
  • Download the java file to run as an application: FunctionID.jar
  • If you download it and run it on your computer as an application, you can resize the window to make the graph bigger or smaller
  • Many of the equations are missing still, because I need to get the list from the organizers of the event.
